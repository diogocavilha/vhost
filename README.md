# Vhost

This script has been created to manage the virtual host files from Apache server and Nginx server.

*It has been only tested in systems Debian-like.*

# Installation

```sh
git clone https://github.com/diogocavilha/vhost.git
cd vhost/
chmod +x vhost.sh
sudo ./vhost.sh
```
